POS order reprint

This module was written to extend the functionality of the point the sale and allow you to print receipt of previously saved orders.

Installation

This module depends on the point_of_sale Odoo official module.

Configuration

No configuration is needed

Usage

To use this module, you need to:

  • launch the point of sale on Odoo
  • on the point of sale user interface, click on the "All orders" tab
  • search your order through the search box
  • click the "Print" button on the selected order line

For further information, please visit:

  • https://www.odoo.com/forum/help-1
